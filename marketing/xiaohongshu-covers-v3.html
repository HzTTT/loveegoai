原理一：情绪链条是怎么形成的

原理科普
一个情绪
是怎么形成的
事件发生 → 大脑判断 → 身体感受
→→ ↓
反复重复 ← 情绪反应 ← 形成链条
↓
性格 → 现实
这条链子一旦形成，身体会跳过大脑直接触发反应
原理科普
为什么身体
比你的决心还快
大脑和身体有一个偷懒机制。

就像你不需要每天重新学走路一样，情绪一旦形成链条，身体就会自动执行。

所以当类似的事件再发生，身体会跳过大脑，直接调动上次的情绪反应。

这不是你的错。这是身体的节能模式。

原理二：头脑和身体为什么不在同一频道

核心概念
DOING 和 BEING
是两个系统
DOING = 头脑、逻辑、分析、计划
BEING = 身体、感受、当下、存在

大多数改变方法在 DOING 层面工作：给你道理、给你方法、给你计划。

但旧的反应模式住在 BEING 层面——在你的身体里。

所以头脑想通了，身体不一定跟。
BEING ≠ 放空 · BEING = 与身体的连接
原理科普
身体感受
和情绪不是一回事
身体感受是客观的：
收缩、扩张、紧、松、热、凉。

情绪是身体感受 + 大脑解读：
"我很愤怒""我很焦虑""我很难过"。

当我们能回到感受本身——只是紧，只是收缩——而不是跳到情绪的故事里，旧链条就少了一个启动环节。
区分感受与情绪

原理三：生存模式 vs 创造模式

原理科普
人的大脑
默认在生存模式
1大脑的首要任务是保护你活下来
2面对不确定，身体进入战斗/逃跑状态
3注意力被锁定在过去（懊悔）和未来（担忧）
4所有能量用于防御，创造力关闭
5你不是在创造人生，是在防御人生
创造只发生在当下。恐惧只存在于时间（过去/未来）。
核心概念
臣服
不是放弃
是退出防御模式
恐惧的根源 =
"我必须掌控一切"

臣服 =
"我允许事情发生"

放下控制 → 神经系统放松
→ 注意力回到当下
→ 进入心流
→ 创造力自然释放

创造力不是努力产生的，
是阻碍消失后的自然流动。

原理四：觉察和练习是怎么起作用的

练习原理
为什么"发出声音
告诉自己"有用
旧程序启动的速度是毫秒级的。

从触发到反应，身体不经过大脑。

但声音有一个物理过程——从喉咙发出，传到耳朵，被意识接收。

这个过程会在触发和反应之间创造一个空间。

有了空间，就有了选择。
有了选择，就不再是自动驾驶。
觉察 → 空间 → 选择
练习原理
回到身体
就是回到当下
头脑可以同时想过去和未来。
但身体只存在于当下。

当你把注意力放在身体上——
感受肩膀是紧还是松，
胸口是扩张还是收缩——

你就自动从头脑的时间线
回到了此刻。

恐惧在时间里。创造在当下。
回到身体 = 回到创造的入口。
身体只在当下

原理五：果因论——先有感受，再有现实

核心概念
不是先有好事
才有好感觉
是反过来的
通常的想法：
等我有钱了 → 我就快乐了
等我找到对的人 → 我就有安全感了

果因论：
先在身体里建立那个感受
→ 你的选择和行动开始不同
→ 现实跟着改变

不是现实给你感受，
是感受创造现实。
原理科普
内化的四个阶段
1不知道且不能 — 还没意识到旧程序的存在
2知道但不能 — 看见了，但还改不了
← 大多数人卡在这里
3知道且能够 — 刻意练习中，能做到
4不知道且能够 — 成为本能，不再需要刻意
从第2到第3阶段需要的是训练，不是更多知识
