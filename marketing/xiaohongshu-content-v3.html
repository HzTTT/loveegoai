#4 头脑和身体为什么不同步（正文页 2-3）

2 / 3
两个系统，两个速度
头脑处理信息的方式是：分析、归纳、得出结论。
身体储存信息的方式是：反复体验，形成自动化反应。
头脑说
"我知道不该生气"
"我想通了"
"下次一定改"
身体说
胸口收紧
心跳加速
跟上次一模一样
头脑可以在一秒内想通。
身体的反应模式是年为单位积累的。
LOVE EGO AI
3 / 3
所以"想通"不等于"做到"
这不是你不够努力。
是你用了头脑的方式去改身体里的东西。
就像你不能用想的方式学会游泳——你必须下水，让身体体验到。
改变反应模式也是一样：
身体要亲自走过一遍新的路径，才会真正记住。
头脑改不了身体
身体要用身体的方式改

#9 觉察到底是什么（正文页 2-3）

2 / 3
觉察不是分析，也不是评判
很多人以为觉察就是"搞清楚为什么"。
觉察不是分析——
"我为什么会这样？因为小时候..."
觉察也不是评判——
"我又这样了，真没用。"
觉察是：
"哦，我现在胸口在收缩。"
只是看到。不解释，不修正。
看到本身
就是改变的起点
LOVE EGO AI
3 / 3
觉察为什么有用
当你觉察到链条在启动的那个瞬间，你和反应之间就出现了一个空间。
没有觉察时：
触发 → 直接反应（自动驾驶）
有了觉察：
触发 → 看到 → 空间 → 选择
有了空间，就有了选择。
有了选择，就不再是自动驾驶。
觉察 → 空间 → 选择

#13 控制消耗能量，臣服释放能量（正文页 2-3）

2 / 3
控制的代价
当大脑感到不确定，第一反应是控制。
控制别人的看法。
控制事情的走向。
控制自己的情绪。
但控制需要大量的能量——你的注意力、身体的紧绷、持续的计算。
这就是为什么你明明没做什么体力活，一天下来特别累。
控制的不是事情
是你自己的能量
LOVE EGO AI
3 / 3
臣服不是放弃
臣服 ≠ 放弃 ≠ 躺平
臣服 = "我允许事情发生"
当你放下控制：
神经系统放松 → 注意力回到当下 → 进入心流 → 创造力自然释放。
创造力不是努力产生的。
是阻碍消失后的自然流动。
放下控制
不是失去力量
是找回力量
